Tienda de Productos

Bienvenidos a nuestro proyecto del equipo B que consume una APIs REST que fue diseñada para gestionar y visualizar información relacionada con usuarios y productos. Este proyecto ha sido desarrollado como parte de un esfuerzo colaborativo utilizando basadas en la web como HTML, CSS y Javascript

El sistema hace uso de JWT (JSON Web Tokens) para la autenticación segura de los usuarios, garantizando que cada solicitud esté protegida mediante tokens encriptados. Express.js, un framework robusto para Node.js, es la base sobre la que hemos construido la API, facilitando el manejo de las rutas y las operaciones CRUD (Crear, Leer, Actualizar, Eliminar).

Para la gestión de datos, hemos implementado MySQL como base de datos, asegurando la integridad y la eficiencia en el almacenamiento de información crítica. Además, el proyecto incorpora el manejo de CORS (Cross-Origin Resource Sharing) para permitir que nuestras aplicaciones front-end interactúen de manera segura con el servidor.

Esta plataforma está diseñada para ser altamente interactiva y fácil de usar. Utilizamos los verbos HTTP como GET, DELETE, PATCH y PUT para realizar operaciones sobre los recursos, permitiendo a los usuarios realizar acciones como consultar la lista de productos, eliminar elementos, actualizar información, y añadir nuevos registros de manera sencilla y efectiva.

En cuanto a la interfaz de usuario, hemos optado por un diseño limpio y moderno utilizando Bootstrap 5 junto con estilos CSS personalizados. Esta combinación nos permite ofrecer una experiencia visual atractiva y responsiva, adecuada para dispositivos de diferentes tamaños, desde smartphones hasta monitores de escritorio.

A través de este proyecto, no solo demostramos nuestras habilidades técnicas en el desarrollo de aplicaciones web, sino que también destacamos la importancia de la seguridad y la usabilidad en la construcción de soluciones tecnológicas que pueden ser escaladas y adaptadas según las necesidades de los usuarios.

Backend

Tecnologías: Express.js, JWT, MySQL, Digital Ocean.

Frontend

Tecnologías: HTML, CSS, JavaScript, Bootstrap 5.

API

Gestión de datos con API REST, implementada usando Node.js y Express. Uso de CORS y Github.
